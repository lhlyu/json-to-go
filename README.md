# json-to-go
将json生成go的数据结构。generating data structure of go languages with JSON.


## 本地开发

> 强烈建议使用 `pnpm` !!!

- 安装 `pnpm install`

- 调试 `pnpm dev`

- 构建 `pnpm build`

- 部署到github `pnpm page`
